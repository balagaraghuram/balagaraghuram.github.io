NHS Oncology Risk Stratification - AI Model Deployment

AI-Driven Risk Stratification in NHS-Funded Oncology Pathways

I worked as a Data Scientist on a collaborative project between NHS hospital trusts and private hospital networks, building a machine learning model to predict deterioration risk in cancer patients after first-line treatment. The gradient boosting model achieved a 0.801 ROC-AUC across 1,842 patients with 87 clinical features covering tumour biology, treatment pathways, comorbidities, acute care utilisation, and longitudinal lab trends. The model is deployed on Google Cloud Run and provides a three-tier risk score (low, medium, high) used by multidisciplinary oncology teams to tailor follow-up intensity and intervene earlier for high-risk patients.

My responsibilities included integrating data from multiple clinical sources, engineering features from structured and longitudinal records, benchmarking four model architectures, performing survival analysis and probability calibration, explaining predictions using SHAP, validating the model through a temporal train-test split, and translating the outputs into a deployable risk stratification tool. The entire pipeline runs in a Docker container on Google Cloud Run with a Python 3.11 backend.
